Platform Vision

Fleet Cloud Platform is a cloud-native platform designed for managing autonomous fleets at scale.

The platform processes telemetry in real time, executes missions, delivers notifications, performs analytics and provides a highly scalable event-driven architecture.

Initially focused on drones, the architecture is designed to support future autonomous systems including warehouse robots, autonomous vehicles and industrial IoT devices.
